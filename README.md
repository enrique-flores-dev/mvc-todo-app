# MVC Todo App
A to-do list web application built with ASP.NET Core MVC and SQLite.

## Features
- User registration and login
- Add, edit, complete, and delete tasks
- Each user has their own private task list

## Technologies
- ASP.NET Core MVC
- Entity Framework Core
- SQLite
- C#

## How to Run
1. Clone the repository
2. Open in Visual Studio
3. Run Update-Database in Package Manager Console
4. Press F5 to run
