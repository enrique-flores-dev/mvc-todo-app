# MVC Todo App
A to-do list web application built with ASP.NET Core MVC and SQLite.

## Features
- User registration and login
- Add, edit, complete, and delete tasks
- Each user has their own private task list
- Email verification on registration
- Forgot password and reset via email

Note: Email sending currently configured for development. 
Full email functionality available after domain verification.

## Technologies
- ASP.NET Core MVC
- Entity Framework Core
- SQLite
- C#
- ASP.NET Core Identity
- Resend (email sending)
- Bootstrap (in progress)

## How to Run
1. Clone the repository
2. Open in Visual Studio
3. Run Update-Database in Package Manager Console
4. Press F5 to run
